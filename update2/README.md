# FocusFriends — Рейтинг + визуальный апгрейд

Это обновление №2, ставится поверх обновления №1 (геймификация).

## Что добавлено

### 2. Рейтинг (страница/вкладка)
- Новая вкладка **«Рейтинг»** на дашборде.
- Использует уже готовый бэкенд `social-service`:
  - `GET /api/leaderboard?period=WEEK|MONTH|ALL` — список игроков и очков.
  - `POST /api/leaderboard/points` — начисление очков.
- Каждое начисление XP (за задачу/привычку) теперь автоматически
  отправляется и в общий рейтинг — можно соревноваться с друзьями.
- Имена и аватары подтягиваются через `GET /api/users/{id}` (auth-service).
- Топ-3 — с медальками, твоя строка подсвечена.

### 3. Визуальный апгрейд
- Тема: градиентная шапка, скруглённые кнопки-«пилюли», более
  насыщенная палитра.
- Иконки категорий у привычек (учёба/спорт/чтение/код/другое) и
  иконка-флажок приоритета у задач.
- Лёгкие микроанимации: элементы списков плавно появляются, карточки
  слегка приподнимаются при наведении.
- Никакой новой функциональности — только внешний вид.

## Установка

```bash
chmod +x install.sh
./install.sh /путь/к/твоему/проекту/focusfriends
cd /путь/к/твоему/проекту/focusfriends
docker compose up -d --build frontend
```

Затем обнови `http://localhost:3000` (лучше жёстко — `Cmd+Shift+R`,
чтобы точно подтянулись новые стили и код).

## Проверка

```bash
cd frontend
npx tsc --noEmit -p tsconfig.app.json
npm run build
```
